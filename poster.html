Satellite Image
Processor

A comprehensive web application for digital image processing — implementing intensity transformations, frequency-domain filtering, image restoration, and color processing algorithms based on Gonzalez & Woods' textbook framework.

Objectives

  • Build an interactive, browser-based platform for real‑time satellite image analysis
  • Implement core DIP algorithms — spatial & frequency-domain transforms
  • Provide side-by-side comparison and quantitative quality metrics (PSNR, SSIM, MSE)
  • Enable noise modelling, restoration, and morphological processing
  • Support multi-space color processing (RGB, HSV, Lab, YCbCr)

Processing Modules

Intensity Transformations
  • Linear, logarithmic & power-law transforms
  • Histogram equalisation & specification
  • Spatial filtering — smoothing & sharpening
  • Edge detection (Sobel, Laplacian, Canny)
Frequency Domain
  • Fourier Transform & spectrum visualisation
  • Low-pass, high-pass & band-pass filters
  • Gaussian, Butterworth & ideal filter kernels
  • Frequency-domain noise removal
Image Restoration
  • Noise modelling — Gaussian, salt-&-pepper, Poisson
  • Wiener filtering & constrained least squares
  • Deblurring & deconvolution techniques
  • Morphological operations (erosion, dilation, opening)
Color Processing
  • Color space conversions (RGB ↔ HSV ↔ Lab ↔ YCbCr)
  • Color enhancement & correction
  • Color-based segmentation
  • Color quantisation & style transfer

Conclusion

The Satellite Image Processor successfully bridges theory and practice by translating the algorithms from Gonzalez & Woods into an accessible, browser-based tool. Its modular Flask + React architecture supports rapid experimentation with intensity, frequency, restoration, and color processing techniques — backed by quantitative metrics for objective evaluation. The project demonstrates a scalable framework for both academic learning and practical satellite image analysis.
